# El Portal Brasileño

Sitio web estático para **El Portal Brasileño**, restaurante de comida rápida premium y saludable en Bogotá.

## ¿De qué trata este proyecto?

La web presenta la marca, el menú y el contacto del negocio con un enfoque comercial y visual moderno.  
El objetivo principal es que el cliente pueda:

- Mostrar sus productos de forma clara y apetecible.
- Recibir pedidos por WhatsApp.
- Reforzar la marca con una presencia más premium y consistente.
- Preparar el terreno para un futuro blog SEO/headless con WordPress.

## Qué incluye hoy

- **Home principal** con hero en video.
- **Menú de productos** con filtros por categoría.
- **Bloque de recetas** como base para contenido SEO.
- **Sección de ubicación** con las dos sedes del negocio.
- **Carrito lateral** y detalle de producto.
- **Botón de pedido por WhatsApp**.

## Estructura del sitio

- `index.html` - página única del sitio.
- `style.css` - estilos globales y responsive.
- `app.js` - lógica vanilla JS para catálogo, carrito y WhatsApp.
- `cms.js` - fuente de datos y soporte de contenido.
- `productos fotos/` - fotos reales de productos, logo y recetas.
- `Fotos producidas/` - video y posters del hero.

## Productos y enfoque comercial

El negocio se organiza en dos grandes grupos:

- **Para comer aquí**: sándwiches, hamburguesas, perros calientes, combos y platos especiales.
- **Para llevar**: carnes frías La Brasileña y productos de deli.

El fuerte del negocio son los **sándwiches** y las **hamburguesas**.  
La línea de **carnes frías** funciona como complemento para llevar a casa.

## Identidad visual

La marca usa una paleta basada en el logo aprobado:

- Amarillo/Oro Portal: `#F2B800`
- Oscuro Elegante: `#1E1E1E`
- Verde Acento: `#008744`
- Rojo Acento: `#E53935`
- Fondo Neutral: `#FFFFFF` / `#FAF9F6`

Tipografías:

- `Outfit` para títulos.
- `Plus Jakarta Sans` para textos.
